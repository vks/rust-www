Installer Rust

Pour installer Rust, lancez la commande suivante dans votre terminal, puis suivez les instructions affichées à l'écran.

curl https://sh.rustup.rs -sSf | sh

Pour installer Rust, téléchargez et exécutez rustup‑init.exe puis suivez les instructions affichées à l'écran.

Notes sur l'installation de Rust

Gestion de chaîne de compilation avec rustup

Rust est installé et géré par l'outil rustup. Rust a un processus de release rapide de six semaines et supporte un grand nombre de plate-formes , alors de nombreuses versions compilées de Rust sont disponibles à tout moment. rustup gère ces versions de manière cohérente sur chaque plate-forme supportée par Rust, permettant l'installation de Rust depuis les canaux de release beta et nightly ainsi qu'un support pour des cibles supplémentaires en cross-compilation.

Pour plus d'informations, consultez la documentation rustup .

Configurer la variable d'environnement PATH

Dans l'environnement de développement Rust, tous les outils sont installés vers le répertoire ~/.cargo/bin %USERPROFILE%\.cargo\bin , et c'est là où vous trouverez la chaîne de compilation Rust, notamment rustc, cargo, et rustup.

Par conséquent, il est d'usage pour les développeurs et développeuses Rust d'inclure ce répertoire dans leur variable d'environnement PATH . Durant l'installation, rustup essaiera de configurer PATH, mais à cause de différences selon les plate-formes, les terminaux de commandes, et de bugs dans rustup, il est possible que ces modifications à PATH ne soient effectives qu'après que la console n'ai été relancée, que l'utilisateur se soit déloggé, ou ne réussissent pas du tout.

Si, après installation, lancer rustc --version dans la console échoue, c'est la raison la plus probable.

Windows considerations

Sur Windows, Rust nécessite également les outils de développement C++ pour Visual Studio 2013 ou plus récent. La manière la plus simple d'obtenir ces outils de développement est d'installer Microsoft Visual C++ Build Tools 2017 qui fournit uniquement les outils de développement Visual C++. Sinon, vous pouvez installer Visual Studio 2017 ou Visual Studio 2015 ou Visual Studio 2013 et sélectionner durant l'installation les « outils C++ ».

Pour plus d'informations sur la façon de configurer Rust sur Windows, consultez la documentation rustup spécifique à Windows.

Autres méthodes d'installation

L'installation décrite ci-dessus, via rustup, est la façon préférable d'installer Rust pour la plupart des développeurs, mais Rust peut aussi être installé via d'autres méthodes.
